BogusExpression Static Watermarking Algorithm

Authors

Balamurgan Chirtsabesan (balamc@cs.arizona.edu)
Tapas Ranjan Sahoo (tapas@cs.arizona.edu)

Description

This algorithm is a very simple watermarker that adds a bogus expression containing a watermark to a classfile.

For example, if expression 987+341 is set to a local variable sm$45 ( local containingprefix 'sm$' ) , then the watermark is the value 1328, we will add the instruction int sm$45 = 987+341 to the constant pool of a class in the user's application.

The watermark value is passed as a parameter, which is split up and an expression is formed using the split values.

Restrictions
The algorithm works only for integer watermarks, and does not support 'Strings' or any other types of watermark.
