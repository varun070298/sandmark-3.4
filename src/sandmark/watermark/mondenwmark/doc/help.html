Monden Watermark Algorithm

Author

Armand Navabi (navabia@cs.arizona.edu)

Description

This static watermarking algorithm implements the watermarking technique described in A Practical Method for Watermarking Java Programs by A. Monden, H. Iida, K. Matsumoto, K. Inoue, and K. Torii. The watermark is embeded by replacing instruction in a dummy method, which is added to the application.

Example

Configuration

No additional configuration parameters are required for embedding and recognition.

References

A. Monden, H. Iida, K. Matsumoto, K. Inoue, and K. Torii, A Practical Method for Watermarking Java Programs. In COMPSAC 2000, 24th Computer Software and Applications Conference, 2000.
